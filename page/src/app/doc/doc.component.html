Page Composition Demo

An Angular app is created with a App Module that contains a number of different components that are visually nested on the display.

The index.html file contains the head and body for the app module. All other HTML content is defined as components that are added to the index page.

Components

index.html contains the following components.

  • app.component
  • header.component
  • footer.component
  • main.component
  • view.content.component

Each component has a TS file that defines the settings used by the components.

A corresponding CSS file defines the styles needed by the component. In this demo we reuse the app.component.css file for each component.

Each component has its own HTML file to be used as a template.
